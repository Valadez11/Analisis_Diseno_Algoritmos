# Instalación de Git

Este repositorio contiene una guía paso a paso para instalar Git y realizar la configuración inicial para comenzar a usar control de versiones y trabajar con GitHub.

---

##  Requisitos

Antes de comenzar asegúrate de tener:

- Una computadora con **Windows, macOS o Linux**
- Conexión a internet
- Permisos para instalar programas

---

# Instalación en Windows

## 1. Descargar Git

1. Ir al sitio oficial:  
   https://git-scm.com  
2. Hacer clic en **Download for Windows**.
3. Se descargará el instalador automáticamente.

---

## 2. Ejecutar el instalador

1. Abrir el archivo descargado.
2. Seguir los pasos del instalador.
3. Puedes dejar las opciones **por defecto**.
4. Presionar **Next** hasta finalizar.

---

## 3. Verificar la instalación

Abrir **Git Bash** o la terminal y ejecutar:

```bash
git --version
